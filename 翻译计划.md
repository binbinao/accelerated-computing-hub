# 文件翻译计划

## 概述
本计划用于将NVIDIA加速计算中心的所有英文内容翻译成中文，并为文件名添加`_cn`标识。

## 文件统计
- 总文件数：90个.ipynb文件
- 主要目录：
  - gpu-python-tutorial: 16个文件
  - gpu-cuda-python-tutorial: 6个文件  
  - gpu-cpp-tutorial: 44个文件
  - Accelerated_Python_User_Guide: 22个文件
  - 其他：2个文件

## 翻译优先级

### 第一阶段：核心教程（高优先级）
1. **gpu-python-tutorial/** - Python GPU开发101教程
2. **gpu-cuda-python-tutorial/** - CUDA Python内核编程
3. **Accelerated_Python_User_Guide/notebooks/** - 最新版用户指南

### 第二阶段：C++教程（中优先级）
1. **gpu-cpp-tutorial/** - 现代CUDA C++基础

### 第三阶段：其他内容（低优先级）
1. **Accelerated_Python_User_Guide/notebooks_v1/** - 旧版用户指南
2. **events/** - 会议资料
3. **learning-pytorch/** - PyTorch学习路径

## 翻译规范

### 文件名规则
- 在文件后缀名之前添加`_cn`
- 示例：`0.0_Welcome.ipynb` → `0.0_Welcome_cn.ipynb`
- 保留原有文件名结构，仅添加语言标识

### 内容翻译原则
1. **技术术语**：保持专业术语的准确性
2. **代码注释**：翻译英文注释，保留代码不变
3. **Markdown内容**：完整翻译说明文字
4. **保持格式**：不改变原有的代码和格式结构

## 实施步骤

1. **创建中文版本**：为每个文件创建`_cn`版本
2. **内容翻译**：翻译文本内容，保持代码不变
3. **质量检查**：确保翻译准确性和一致性
4. **更新索引**：更新README文件中的链接

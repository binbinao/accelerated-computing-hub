# GPU加速计算学习计划

## 项目概述

本仓库是NVIDIA官方的GPU加速计算学习中心，包含从入门到高级的完整教程体系。

## 学习路径建议

### 路径一：Python开发者（推荐初学者）

#### 阶段1：GPU计算基础（1-2周）
- **教程**: `gpu-python-tutorial/`
- 内容：
  - GPU与CPU的区别
  - CUDA开发模型概述
  - Numba编译器基础
  - 编写第一个GPU Python代码
- 建议顺序：
  1. `0.0_Welcome.ipynb` - 欢迎和概述
  2. `1.0_CPU_GPU_Comparison.ipynb` - CPU与GPU对比
  3. `2.0_Numba.ipynb` - Numba基础
  4. `2.1_Numba_lab.ipynb` - 实验练习

#### 阶段2：GPU数据科学（2-3周）
- **教程**: `gpu-python-tutorial/` 继续
- 内容：
  - CuPy科学计算库
  - cuDF数据处理
  - cuML机器学习
  - 多GPU与Dask集成
- 建议顺序：
  1. `5.0_Cupy.ipynb` - CuPy基础
  2. `6.0_cuDF.ipynb` - GPU数据框
  3. `7.0_cuML.ipynb` - GPU机器学习
  4. `8.0_Multi-GPU_with_Dask.ipynb` - 多GPU编程

#### 阶段3：CUDA Python内核编程（2周）
- **教程**: `gpu-cuda-python-tutorial/`
- 内容：
  - CUDA Python生态系统
  - 内核编程基础
  - 并行计算模式
- 建议顺序：
  1. `0.0_Welcome.ipynb`
  2. `1.0_CUDA_Core.ipynb`
  3. `2.0_CUDA_CCCL_Parallel.ipynb`

### 路径二：C++开发者

#### 阶段1：现代CUDA C++基础（3-4周）
- **教程**: `gpu-cpp-tutorial/`
- 模块1：并行算法加速
  - 执行空间概念
  - 算法扩展
  - 内存空间管理
- 模块2：异步和CUDA流
  - 异步编程
  - 流管理
  - 固定内存
- 模块3：CUDA内核实现
  - 内核编程
  - 原子操作
  - 共享内存
  - 协作算法

### 路径三：综合学习（推荐有经验开发者）

#### 阶段1：加速Python用户指南（4-6周）
- **教程**: `Accelerated_Python_User_Guide/`
- 完整章节：
  1. GPU计算基础
  2. CUDA简介
  3. Python在GPU上的应用
  4. 使用CuPy进行科学计算
  5. 使用Numba编写CUDA内核
  6. nvmath-python简介
  7. cuDF简介
  8. cuML简介
  9. cuGraph简介
  10. 开发者工具
  11. 分布式计算
  12. NVIDIA Warp简介

- 免费T4 GPU即可满足学习需求
- 每个notebook都有对应的实验和解决方案

- 会议资料：`events/` 包含GTC、PyCon、SciPy等会议内容
- PyTorch学习：`learning-pytorch/`

## 学习建议

### 时间安排
- **初学者**：建议每天1-2小时，持续2-3个月
- **有经验开发者**：可以加速学习，重点关注核心概念
- **项目实践**：每个阶段完成后尝试小型项目应用

### 学习技巧
1. **动手实践**：所有教程都包含可运行的代码示例
2. **循序渐进**：按照建议顺序学习，不要跳跃
3. **理解概念**：重点关注GPU计算的核心思想
4. **社区参与**：遇到问题时使用NVIDIA开发者论坛

### 进阶方向
- 深度学习：结合PyTorch学习路径
- 高性能计算：深入学习CUDA C++
- 数据科学：掌握RAPIDS生态系统
- 分布式计算：学习多GPU和集群编程

## 评估标准

- **基础掌握**：能够理解GPU计算的基本概念
- **中级水平**：能够使用Python库进行GPU加速计算
- **高级水平**：能够编写自定义CUDA内核和优化算法

## 持续学习

- 关注`events/`目录中的最新会议资料
- 定期查看NVIDIA官方文档更新
- 参与社区讨论和项目贡献
